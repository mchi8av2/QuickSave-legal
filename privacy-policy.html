QuickSave Privacy Policy

Effective date: 31 March 2026

QuickSave ("we", "our", "us") provides a Chrome extension that helps users compare grocery prices and improve checkout decisions across supported shopping platforms.

This Privacy Policy explains what data is processed, why it is processed, and your choices.

1. Data We Process

QuickSave processes only data required for its core functionality:

  • Shopping page context from supported domains
  • User-provided inputs required for comparison features (for example, location input)
  • Basic technical request metadata generated by infrastructure (standard server logs)

2. Why We Process Data

We process data only to provide the extension's single purpose:

  • Compare shopping data across supported grocery services
  • Return price and availability comparison results
  • Improve reliability and performance of the service

We do not process data for unrelated purposes.

3. Data Sharing and Transfers

Data is transmitted to:

  • QuickSave backend API (to compute comparison results)
  • Limited third-party utility services required for feature operation (for example, geocoding lookups)

We do not sell personal data. We do not share data for personalized advertising.

4. Chrome Permissions and Use

QuickSave requests only permissions needed for core functionality:

  • storage: save extension state and preferences required for operation
  • scripting: run extension logic on supported shopping pages
  • tabs: interact with the currently active supported tab
  • Host permissions:
    • Blinkit domains (read relevant shopping context on supported pages)
    • Zepto domains (read relevant shopping context on supported pages)
    • QuickSave API domain (send requests and receive comparison responses)
    • Nominatim OpenStreetMap API (location/geocoding support when required)

5. Data Retention

We keep data only as long as needed for service operation, security, debugging, and legal compliance. We aim to minimize retained data whenever possible.

6. Security

We use reasonable technical and organizational measures to protect data in transit and at rest. No system is perfectly secure, but we continuously work to reduce risk.

7. Your Choices

  • Stop data processing by disabling or uninstalling the extension
  • Avoid using features that require optional user input

8. Children's Privacy

QuickSave is not directed to children under 13, and we do not knowingly collect personal data from children.

9. Changes to This Policy

We may update this policy from time to time. We will post the updated version with a revised effective date.
